Your Personal Website Checklist

A step-by-step guide to consolidate and launch your site on GitHub Pages.

This special repository will be the foundation of your entire website. Naming it correctly is the most important part.

  1. Navigate to GitHub and click New repository.
  2. For the repository name, you must use the exact format: your-username.github.io.
  3. Set the repository to Public.
  4. Check the box to Add a README file.
  5. Click Create repository. You now have the home for your website!

Clone the new repository to your computer and create the necessary files and folders. This structure will keep your project organized.

First, open your terminal or command prompt and run this command:

git clone https://github.com/your-username/your-username.github.io.git

Next, inside the new folder, create the following structure:

With your files created and content added, it's time to send them to GitHub. This process is called "pushing" and it makes your website live on the internet.

In your terminal, inside your project folder, run these three commands one by one:

# 1. Stage all your new files for commit git add .  # 2. Create a commit with a descriptive message git commit -m "Build initial version of website"  # 3. Push your changes to GitHub git push

Your website is now live! Here are the final steps to complete the consolidation.

  1. Visit your new website at https://your-username.github.io. It may take a minute to appear.
  2. Click through every button and link to ensure they all work as expected.
  3. Go to your main GitHub profile repository (`your-username/your-username`) and edit the `README.md` to point to your new site URL.
  4. To prevent confusion, Archive your old repositories via their "Settings" tab.
